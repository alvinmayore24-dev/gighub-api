\# Nairobi Freelance Gigs API



Admission Number: C027-01-0881/2024



This project implements a REST API for managing freelance gig listings for a Nairobi-based freelancing platform called GigHub.



\## Features



\* List all gigs

\* Search for gigs by title

\* View gig details by ID

\* Create new gigs

\* Update gig budgets and status

\* Delete gigs
